# Drum-Kit-Challenge
I created an interactive Drum Kit web application using HTML, CSS, and JavaScript, allowing users to play different drum sounds by either clicking on buttons or pressing corresponding keys on the keyboard. This project showcases my ability to combine markup, styling, and scripting to create an engaging user experience. I used JavaScript to handle event listeners, play audio clips, and provide visual feedback through simple animations. Through this project, I gained hands-on experience in DOM manipulation, event handling, and enhancing UI responsiveness—key skills in modern front-end development.
