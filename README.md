# 🥁 Drum Kit

An interactive **Drum Kit Web Application** built using **HTML, CSS, and JavaScript** that allows users to play different drum sounds using keyboard keys or mouse clicks.

## 📌 Project Overview

Drum Kit is a frontend project designed to demonstrate interactive web development concepts by creating a virtual drum experience.

Users can trigger different drum sounds through:

* Keyboard key presses
* Button clicks

The project uses JavaScript to handle user interactions, play audio files, and provide visual feedback.

## ✨ Features

🥁 Interactive drum buttons
🎹 Keyboard-based controls
🔊 Real-time audio playback
✨ Button animation effects
⚡ Dynamic user interaction
📱 Responsive interface design

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript
